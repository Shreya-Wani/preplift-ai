# 🚀 PrepLift AI

PrepLift AI is a full-stack GenAI-powered job preparation platform designed to help users streamline their interview readiness. It allows users to analyze resumes, identify skill gaps, and generate AI-driven interview questions along with ATS-friendly resumes.

The platform integrates modern web technologies with AI capabilities to simulate a real-world career preparation system, making it both practical and scalable.

---

## ✨ Features
- Resume upload and intelligent analysis  
- AI-powered skill gap detection based on job descriptions  
- Personalized interview question generation  
- ATS-optimized resume creation  
- Secure authentication using JWT and token handling  
- Dynamic PDF generation using Puppeteer  

---

## 🛠 Tech Stack
Frontend: React.js, Vite  
Backend: Node.js, Express.js  
Database: MongoDB  
AI Integration: Gemini API  
Other Tools: Puppeteer, Multer  

---

## 🎯 Purpose
This project is built to demonstrate a production-level full stack application with GenAI integration, focusing on solving real-world problems in job preparation and career development.
